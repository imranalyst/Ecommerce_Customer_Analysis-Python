# E-commerce Customer Behavior Analysis using Python

## Project Overview

This project focuses on analyzing customer purchasing behavior, customer satisfaction, retention patterns, payment preferences, and delivery performance in an e-commerce platform using Python-based data analysis techniques.

The main objective of this project is to extract meaningful business insights from customer transaction data and understand the factors that influence customer engagement, purchasing decisions, and repeat purchase behavior.

The project was completed using Exploratory Data Analysis (EDA), statistical analysis, and data visualization techniques to transform raw customer data into actionable business insights.

---

## Objectives

The key objectives of this project are:

* Analyze customer purchasing behavior and spending patterns
* Understand customer satisfaction and review behavior
* Identify factors influencing return customers
* Analyze payment method preferences
* Evaluate delivery performance across subscription types and locations
* Measure the impact of discounts on customer spending
* Discover high-performing product categories and regions
* Generate business insights for better decision-making

## Project Workflow

### 1. Data Preprocessing

* Checked missing values
* Verified duplicates
* Reviewed dataset structure
* Prepared data for analysis

### 2. Exploratory Data Analysis (EDA)

Performed statistical and visual analysis to identify:

* Customer demographics
* Spending behavior
* Product preferences
* Satisfaction patterns
* Retention behavior

### 3. Data Visualization

Created multiple visualizations including:

* Bar Charts
* Histograms
* Countplots
* Heatmaps
* Boxplots

### 4. Business Insight Generation

Generated data-driven business insights and strategic recommendations based on customer behavior analysis.

---

## Key Findings

* Middle-aged customers represented the dominant customer segment
* Customers who spent more time on the website tended to purchase more products
* Discounts positively influenced customer spending behavior
* Mobile devices were the most frequently used purchasing platform
* Premium subscribers generally experienced faster delivery services
* Customer satisfaction strongly influenced repeat purchasing behavior
* Payment convenience affected customer satisfaction and retention
* Purchase behavior and delivery performance varied across locations
